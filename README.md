# Calculator Pro - GitHub Pages Deployment

This repository hosts the static build of **Calculator Pro** at [https://rahulbisht.github.io](https://rahulbisht.github.io).

## About

This repository contains the production build artifacts from the [calculator](https://github.com/RahulBisht/calculator) project. The files are automatically generated and deployed using the deployment script.

## Deployment

The site is automatically deployed from the calculator project using the `deploy.sh` script. 

**Do not manually edit files in this repository** - all changes should be made in the source calculator repository and then deployed using the deployment script.

## Technology Stack

- **Framework**: Next.js (Static Export)
- **Hosting**: GitHub Pages
- **Domain**: rahulbisht.github.io

## Last Updated

This site is automatically updated whenever the deployment script is run from the calculator project.
